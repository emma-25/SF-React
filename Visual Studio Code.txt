Help, ayuda de terminal, tiene varias options que nos ayuda para trabajar en consola.
ls, (ls -l, ls -a)listar lo que tenemos en nuestra carpeta.
cd,(cd -, cd ., cd .., cd /, cd c, cd ../..)cambiar de directorio.
cd 'Program Files', cuando las carpetas tienen espacios en blancos, poner el nombre entre comillas.
touch, crear archivo 'hola.txt'.
eco 'Hola, soy Emmanuel' > saludo.txt, crear archivo con contenido.
cat saludo.txt, podemos ver el contenido.
mkdir, crear carpetas.
rmdir 'assets', eleminar carpetas vacias.
rm 'hola.txt', eliminar archivos.
rm -r, elimina carpetas con archivos. 
rm -rf, rmeliminar carpetas de forma dorzada. (rm --help).
mv 'hola.txt assets' , mover archivo a otra carpeta.
mv 'hola.txt script.txt', po demos renombrar el nombre a un archivo.
cp hola.txt ../, copiar un archivo y mover a otra carpeta.
cp hola.txt ../nuevo.txt, copiar el contenido del archivo en un nuevo archivo.
cp -r js css, copiar carpeta con contenido a otra carpeta.
find, (find j*) buscar archivos o varios archivos que comiencen con 'j', por ejemplo.
ps,listar proceso de nuestra pc.
kill + id, matar el proceso.
nano(ctrl + x, matar el proceso), vim.
